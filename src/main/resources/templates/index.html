Framework Features

Feature Management
Create, edit, and manage Cucumber feature files with a user-friendly interface. Support for scenarios, backgrounds, and examples.
Tag Management
Organize tests with custom tags. Execute tests based on tags with AND/OR logic. Track tag usage and statistics.
Test Execution
Execute tests individually or by tags. Real-time progress tracking with WebSocket. Support for parallel execution.
Page Object Manager
Create and manage page objects with web elements. Auto-generate page object classes. Support for multiple locator strategies.
Step Definitions
Generate step definition templates automatically. Manage and organize step definitions. Support for custom parameters.
Test Data Manager
Manage test data in JSON format. Environment-specific data sets. Easy data retrieval and updates.
Execution History
View complete test execution history. Filter by tags, dates, and status. Track test trends over time.
Test Reports
View detailed Cucumber HTML reports. Download reports for sharing. Visual analytics and insights.
Documentation
Access comprehensive framework documentation. Learn best practices. View Cucumber and BDD guides.
